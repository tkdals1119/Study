## 알고리즘
```
> 그리드 알고리즘(탐욕 알고리즘, 욕심쟁이 알고리즘)
  - 간단히 말해서 최적의 해를 구하기 위해서는 첫 번째 활동이 최대한 일찍 끝나면 된다.
    - 오름 차순과 비슷하다고 보면 될 것 같다.

  - if(arr[i] > arr[i+1])
    {
      temp = arr[i];
      arr[i] = arr[i+1];
      arr[i+1] = temp;
    }
```
